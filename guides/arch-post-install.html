Window display with X.org

Install the following packages

xorg-server
xorg-xinit
libxinerama
libxft

DWM, ST, DMenu

$ git clone https://git.suckless.org/dwm
$ git clone https://git.suckless.org/st
$ git clone https://git.suckless.org/dmenu
$ git clone https://git.suckless.org/tabbed

Install each cloned repo with $ sudo make clean install

Create a xinitrc file in $HOME$ touch .xinitrc

Edit the file with

exec dwm

Extras (wallpaper & X.org utilities)

xorg-xset
xorg-xsetroot
xwallpaper

Now you should be able to start with $ startx

Audio and Bluetooth

Audio

$ sudo pacman -S pulseaudio pulsemixer

Bluetooth

$ sudo pacman -S bluez bluez-utils pulseaudio-bluetooth

Start / Enable services:

sudo systemctl start bluetooth.service
sudo systemctl enable bluetooth.service
