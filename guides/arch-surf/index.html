Arch SURF Browser

Go Back…

Clone Surf

$ git clone https://git.suckless.org/surf

Install the following dependencies.

$ pacman -S webkit2gtk gcr

Compile Surf

$ sudo make clean install

Extra packages for surf

By default surf cant play youtube videos. Your system will need the following packaged for it to work.

$ pacman -S gstreamer gst-plugins-good gst-libav gst-plugins-base xdg-utils
