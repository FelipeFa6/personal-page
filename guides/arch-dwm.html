dwm & dmenu

Install the following dependencies

$ pacman -S xorg-server xorg-xini libxinerama libxft webkit2gtk

Clone dwm & dmenu

$ git clone https://git.suckless.org/dwm
$ git clone https://git.suckless.org/dmenu

Compile the cloned repos with

$ sudo make clean install

Create a X config file

$ echo "exec dwm" > ~/.xinitrc

Wallpaper

Static Wallpaper (xsetroot)

For minimalism elitists there is a perfect option for a wallpaper with solid colors

Install the xsetroot utility

$ pacman -S xorg-xsetroot

And run the following command

$ xsetroot -solid COLOR

Image Wallpaper (feh)

I use feh because is a image viewer that has the hability to set a wallpaper
(2 in 1 = less bloat).

$ pacman -S feh

# Set a random image in a folder
$ feh --bg-scale --randomize ~/dir/to/wallpaper/* --no-fehbg

# Set a single image
$ feh --bg-scale ~/dir/to/wallpaper
