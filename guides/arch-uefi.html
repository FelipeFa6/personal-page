Go Back...

Should be connected to the internet via cable, for wifi refer to the arch wiki.

Sync time protocol

$ timedatectl set-ntp true

Partitions Get disks

$ lsblk

Enter your hardware storage

$ fdisk /dev/sdx

Partition the disks as you like.

For this tutorial i will be doing 2 partitions

UEFI partition -> 200MB -> EFI System (press t and then 1 for EFI) Full storage -> Remaining size.

Format partitions EFI Partition

$ mkfs.fat -F32 /dev/sdx
$ mkfs.ext4 /dev/sdx

Mount of partitions

Storage partition

$ mount /dev/sdx /mnt

EFI partition

$ mkdir /mnt/boot/
$ mount /dev/sdx /mnt/boot

Installing base system

$ pacstrap /mnt base base-devel linux-lts linux-firmware vim git

For WiFi Internet add: dhpcd iwd

For Wired Internet add: networkmanager

Generate FSTAB

$ genfstab -U /mnt >> /mnt/etc/fstab

Change installation medium

$ arch-chroot /mnt

Create SWAP file.

$ fallocate -l 2GB /swapfile
$ chmod 600 /swapfile
$ mkswap /swapfile
$ swapon /swapfile

Change FSTAB for SWAP file

$ vim /etc/fstab

Inside the FSTAB file add the following:

/swapfile none swap defaults 0 0

Timezone

$ ln -sf /usr/share/zoneinfo/Chile/Continental /etc/localtime
$ hwclock --systohc

Change locale.gen

Edit the /etc/locale.gen file and uncomment your language, in my case is en_US-UTF-8 UTF-8

Enter the following command:

$ locale-gen

Edit locale.conf

$ vim /etc/locale.conf

Add hostname

$ vim /etc/hostname

Set your prefered name my case is t4rkf

Add hosts

Edit /etc/hosts:

127.0.0.1   localhost
::1   localhost
127.0.1.1   userName.localdomain   userName

Add a password to your root

$ passwd

Set the bootloader

Download following software

$ pacman -S grub efibootmgr

GRUB install

$ grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=GRUB
$ grub-mkconfig -o /boot/grub/grub.cfg

Add new user

$ useradd -mG wheel newUser
$ passwd newUser

Edit group 'wheel' permisions

$ visudo

Reboot system
