Go Back...

Should be connected to the internet via cable, for wifi refer to the arch wiki.

Sync time protocol

$ timedatectl set-ntp true

Partitions Get disks

$ lsblk

Enter your hardware storage

$ fdisk /dev/sdx

Partition the disks as you like.

For this tutorial i will be doing 2 partitions

- SWAP partition with 10GB
- Storage partition with remaining space. (Remember to put the BOOT flag with 'a' in fdisk)

Format the partitions

This is the storage partition

$ mkfs.ext4 /dev/sdx

Format SWAP partition.

$ mkswap /dev/sdxN

Mount of partitions

Storage partition

$ mount /dev/sdxN /mnt

Mount SWAP partition

$ swapon /dev/sdx

Installing base system

$ pacstrap /mnt base base-devel linux-lts linux-firmware vim git networkmanager

Generate FSTAB

$ genfstab -U /mnt >> /mnt/etc/fstab

Change installation medium

$ arch-chroot /mnt

Enable Networkmanager

$ systemctl enable NetworkManager

Timezone

$ ln -sf /usr/share/zoneinfo/Chile/Continental /etc/localtime
$ hwclock --systohc

Change locale.gen

Edit the /etc/locale.gen file and uncomment your language, in my case is en_US-UTF-8 UTF-8

Enter the following command:

$ locale-gen

Edit locale.conf

$ vim /etc/locale.conf

Add LANG=en_US.UTF-8

Add hostname

$ vim /etc/hostname

Set your prefered machine name, in my case is t4rkf

Add hosts

Edit /etc/hosts:

127.0.0.1   localhost
::1   localhost
127.0.1.1   userName.localdomain   userName

Add a password to your root

$ passwd

Set the bootloader

Download following software

$ pacman -S grub

GRUB install

Do this to the whole device where Arch is installed

$ grub-install /dev/sdx
$ grub-mkconfig -o /boot/grub/grub.cfg
$ mkinitcpio -P

Add new user

$ useradd -mG wheel newUser
$ passwd newUser

Edit group 'wheel' permisions

$ visudo

Reboot system
